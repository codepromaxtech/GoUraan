### Color Schemes

#### Flight Page (Blue Theme)
```css
Primary: Blue 600 (#2563eb)
Secondary: Blue 700 (#1d4ed8)
Gradient: from-blue-600 to-blue-800
Accent: Blue 500 (#3b82f6)
```

#### Hotel Page (Purple/Pink Theme)
```css
Primary: Purple 600 (#9333ea)
Secondary: Pink 600 (#db2777)
Gradient: from-purple-600 via-pink-600 to-red-600
Accent: Purple 500 (#a855f7)
```

### Typography
- **Headings**: 4xl-5xl, bold
- **Subheadings**: 3xl, bold
- **Body**: Base, medium
- **Labels**: sm, medium

### Components
- **Cards**: Rounded-xl, shadow-md, hover:shadow-2xl
- **Buttons**: Gradient, rounded-lg, shadow-lg
- **Inputs**: Border-2, rounded-lg, focus:ring-2
- **Icons**: SVG, 5-8 size, colored
